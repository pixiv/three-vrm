Optional renderOrderOffset

renderOrderOffset?: number
  • Defined in packages/three-vrm-materials-mtoon/src/MToonMaterialLoaderPluginOptions.ts:9

This value will be added to every meshes who have MaterialsMToon. The final renderOrder will be sum of this renderOrderOffset and renderQueueOffsetNumber for each materials. 0 by default.

Optional v0CompatShade

v0CompatShade?: boolean
  • Defined in packages/three-vrm-materials-mtoon/src/MToonMaterialLoaderPluginOptions.ts:17

There is a line of the shader called "comment out if you want to PBR absolutely" in VRM0.0 MToon. When this is true, the material enables the line to make it compatible with the legacy rendering of VRM. Usually not recommended to turn this on. false by default.
