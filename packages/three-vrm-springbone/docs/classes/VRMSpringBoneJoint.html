Class VRMSpringBoneJoint

A class represents a single joint of a spring bone. It should be managed by a [[VRMSpringBoneManager]].

Private _worldSpaceBoneLength

_worldSpaceBoneLength: number = 0.0

Length of the bone in world unit. Will be used for normalization in update loop, will be updated by _calcWorldSpaceBoneLength.

It's same as local unit length unless there are scale transformations in the world space.
