Static from

  • from(gltf: GLTF, options?: VRMImporterOptions): Promise<VRM>
    • Defined in packages/three-vrm/src/VRM.ts:53

    Create a new VRM from a parsed result of GLTF taken from GLTFLoader. It's probably a thing what you want to get started with VRMs.

    example

    Most basic use of VRM

    const scene = new THREE.Scene();

new THREE.GLTFLoader().load( 'models/three-vrm-girl.vrm', ( gltf ) => {

  THREE.VRM.from( gltf ).then( ( vrm ) => {

    scene.add( vrm.scene );

  } );

} );

    Parameters

    • gltf: GLTF

      A parsed GLTF object taken from GLTFLoader

    • options: VRMImporterOptions = ...

      Options that will be used in importer

    Returns Promise<VRM>
